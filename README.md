# CISM-Practice-Questions
CISM Practice Questions in a Quiz Format to support preparation for the CISM Exam Jan 2026

The Quiz is customisable by the end user.

Quiz features:
-Domains can be selected — choose 1, 2, 3 or all 4 domains
-3 quiz modes: Exam Weighted (mirrors real CISM ratios: 17/20/33/30%), Equal Mix, or Domain Only
-Adjustable question count — slider from 10 to 150
-Per-domain score breakdown on the results screen
-The explanation for the correct answer (highlighted in green)
-The explanation for your wrong choice if applicable (highlighted in red)
-Final score screen with percentage, breakdown, and a performance message
-Instant feedback after each answer — shows correct/incorrect with the right answer highlighted
-Live score displayed throughout
